Backend Assessment 

1. Create an express based node.js server with below requirements. The API should accept names of 
multiple cities as input and fetch real time weather from any weather API's and respond with real time weather results.
 Create a POST endpoint /getWeather which accepts the name of the cities as POST body params 
 Example Input { "cities": [ "toronto", "mumbai", "london" ] } 
 Example Output { "weather": { "toronto": "24C", "mumbai": "34C", "london": "14C" } } 

2. Bonus Create a simple UI app which accepts the names of the cities as input in a text box and makes a call to the 
API endpoint created in #1 and also displays the result in the UI.
